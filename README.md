# Aetherium-From-Explicit-Physics-to-Neural-World-Models
Aetherium World Model : Un projet de recherche explorant la frontière entre la mécanique des fluides et le Deep Learning. Ce dépôt implémente une simulation physique déterministe (NumPy) et son modèle de monde prédictif (PyTorch) capable d'apprendre des dynamiques latentes factorisées.
